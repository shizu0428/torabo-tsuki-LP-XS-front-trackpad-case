# torabo-tsuki-LP-XS-front-trachpad-case
# torabo-tsuki LP XSサイズ用 拡張トラックパッド手前配置カバー

このリポジトリは、せきごん氏（sekigon-gonnoc）が販売・公開しているキーボード「**torabo-tsuki LP XSサイズ**」に、「**拡張トラックパッド**」を追加するためのカスタムカバー（ケース）の3D設計データを公開しています。

---

## 📌 概要・設計コンセプト

本プロジェクトのカバーは、せきごん氏が公開されている「XSサイズ用標準カバー」のデータをベースに設計しています。

* **トラックパッドへのアクセス向上:** マイコン（BMP Boost）の手前側キー1つをオミットすることで、トラックパッドにアクセスしやすくなることを意図しています。四角で囲われた範囲の裏側にトラックパッドを配置します。
* 左右それぞれのデータを準備しています。

## 🛠️ 組み立て方法と注意点

基本的な組み立て方法や注意点については、上記の公式ビルドガイド（ミニトラックパッドオプションの組み立て方）に準拠します。

| 製造方式 | 動作確認状況 / 推奨設定 |
| :--- | :--- |
| **FDM方式（家庭用3Dプリンター等）** | 100%未満で作成した場合、強度不足や構造上の理由から正しく動作しない可能性が非常に高いです。 |
| **SLA方式（光造形 / JLCPCB等）** | JLCPCBのSLA方式（レジン等）で製造したパーツにおいては、問題なく動作することを確認しています。 |
